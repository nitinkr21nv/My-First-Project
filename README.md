# Quad AI School of Technology and Management Website

This project is a multi-page educational website built using HTML and CSS.
It is designed for Quad AI School of Technology and Management to provide information about courses, faculty, workshops, and contact details.

## Features
- Home Page
- Courses Page
- Faculty Page
- Contact Page
- Simple and Clean UI

##Technologies Used
- HTML
- CSS

## How to Run 
1. Download the project
2. Open index.html in any browser.
